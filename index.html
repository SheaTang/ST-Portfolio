Portfolio 

MIT Portfolio Project 

PacMen Factory 

An interactive PacMen Factory at the tip of your fingers; this game makes use of many callback functions. In the "game" many PackMen are moving and bouncing off-of the walls, each at different speeds. The user, can add as many PacMen as they wish by clicking the button functions in the upper right corner Add PacMan and Start Game. A full set of instruction on how to download and run can be found by clicking the PacMan photo (located above).

Eye Movement Tracker 

An interactive way to play with the shadow Dom with the click of your mouse. The Eye Movement Exercise is a great depiction of what is possible when HTML , CSS , and JavaScript come together. The exercise creates a webpage with two synchronized eyes that follow the user’s mouse movement. The Eye Movement exercise is a Mouse event that binds DOM elements to a "mouse event".

Real Time Bus Tracker 

In this project, data is derived from the MBTA (Massachusetts Bay Transportation Authority) API, in order to track the real time bus movement of: Bus 1, that travels between Harvard Square and Nubian Station (both located within the City of Boston).
